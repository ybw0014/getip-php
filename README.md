# GetIP

## Description

getip-php is a PHP library for getting the IPv4 address of client.

* License: MIT, see [LICENSE](https://github.com/ybw0014/getip-php/blob/main/LICENSE)

## Installation

```sh
composer require ybw0014/getip-php
```

## Usage

```php
<?php
use ybw0014\IP;

$ip = IP::get();
```
